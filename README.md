# LED Blink Project 💡

A simple beginner project using Arduino to blink an LED on and off every 1 second.

## 🔧 Components
- Arduino Uno (or any compatible board)
- 1x LED
- 220Ω resistor (optional)
- Breadboard + jumper wires
